A Django plugin for iOS StoreKit server.

## Supported python versions

3.6.x ~ 3.9.x

## Supported django versions

2.x

## Installation

```shell script
$ pip install django-ios-storekit
```

Add `storekit` into `INSTALLED_APPS` in `settings.py` file.

```python
INSTALLED_APPS += (
    'storekit',
)
```

```shell script
$ python manage.py migrate
```
